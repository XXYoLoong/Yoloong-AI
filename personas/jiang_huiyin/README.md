# 江徽音人格资产

本目录由用户提供的两份人设文档抽取和整理而来：

- `0.0.1.docx`
- `长.docx`

运行时优先加载：

- `SOUL.md`：核心人格和关系底色。
- `STYLE.md`：语言风格与情绪表达。
- `BOUNDARIES.md`：边界、风险和禁止倾向。
- `MEMORY_SCHEMA.md`：需要长期记住的用户信息类型。

这些文件是稳定摘要，不直接保存原始 `.docx` 全文。
